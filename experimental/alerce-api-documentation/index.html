Introduction

The ALeRCE broker provides two parallel API backends for accessing processed astronomical alert data.

The ZTF v1 API (api.alerce.online/ztf/v1/) serves data from the Zwicky Transient Facility. It uses a RESTful, path-based endpoint structure where the object identifier is embedded in the URL path (e.g. /objects/{oid}/lightcurve).

The LSST Multisurvey API (api-lsst.alerce.online/) serves data from the Vera C. Rubin Observatory's Legacy Survey of Space and Time. It uses an independently-deployed microservice architecture with query-parameter-based object selection (e.g. /lightcurve_api/lightcurve?survey_id=lsst&oid=X).

Both APIs are read-only (HTTP GET), require no authentication, and return JSON. This documentation covers both backends, noting differences where they diverge.

Source code: github.com/alercebroker/web-services. Python client: alerce_client.

Conventions

Photometric units. LSST returns flux in nanojansky (nJy) via psfFlux / scienceFlux. ZTF returns AB magnitudes via magpsf / sigmapsf. To convert ZTF magnitudes to nJy: flux = 10^((31.4 − magpsf) / 2.5).

Band encoding. LSST uses a band integer with a band_map dictionary ({"0":"u", "1":"g", "2":"r", "3":"i", "4":"z", "5":"y"}). ZTF uses an integer fid field: 1 = g, 2 = r, 3 = i.

Time system. All times are Modified Julian Dates (MJD) in UTC.

Range filters. Parameters such as n_det/ndet and firstmjd accept repeated values to define a range: first value = minimum, second = maximum. Example: ?firstmjd=60000&firstmjd=60100.

64-bit identifiers. LSST oid and measurement_id values are 64-bit integers that exceed JavaScript's Number.MAX_SAFE_INTEGER. Parse them as strings in JavaScript. ZTF OIDs are short alphanumeric strings (e.g. ZTF21aaqjmkl) and candid values are also large integers.

Pagination. Both APIs support page and page_size. LSST wraps results in {"items": [...]}. ZTF returns a flat array.

ZTF forced photometry uses magnitude + sign convention. The isdiffpos field (1 = positive, 0 or −1 = negative) determines the sign of the difference flux. To convert: flux = sign × 10^((31.4 − mag) / 2.5). Forced photometry records lack magpsf; they use mag/e_mag instead.

Classification

Probabilities and classifier metadata
GET /probability_api/probability · /objects/{oid}/probabilities Per-class probabilities for an object ▾

Returns classification probabilities from all classifiers that have been run on the object. Each record contains a classifier name, version, class label, and probability. Probabilities from a given classifier + version sum to 1.0.

Python Quick Start

Minimal examples using requests. For a richer client, see alerce_client.

Search for SN candidates (LSST)
import requests resp = requests.get("https://api-lsst.alerce.online/object_api/list_objects", params={ "survey": "lsst", "classifier": "stamp_classifier_rubin_beta", "class_name": "SN", "page_size": 20, "count": "false", }) objects = resp.json()["items"] for obj in objects: print(obj["oid"], obj["meanra"], obj["meandec"], obj.get("probability"))
Search for SN candidates (ZTF)
import requests resp = requests.get("https://api.alerce.online/ztf/v1/objects", params={ "classifier": "stamp_classifier", "class": "SN", "ranking": 1, "page_size": 20, }) objects = resp.json() # flat array, no "items" wrapper for obj in objects: print(obj["oid"], obj["meanra"], obj["meandec"])
LSST light curve (flux)
import requests oid = "2457648302395060224" data = requests.get(f"https://api-lsst.alerce.online/lightcurve_api/lightcurve", params={"survey_id": "lsst", "oid": oid}).json() for det in data["detections"]: band = det["band_map"][str(det["band"])] print(f"{det['mjd']:.4f} {band} {det['psfFlux']:.1f} ± {det['psfFluxErr']:.1f} nJy")
ZTF light curve (magnitudes)
import requests oid = "ZTF21aaqjmkl" data = requests.get(f"https://api.alerce.online/ztf/v1/objects/{oid}/lightcurve").json() fid_map = {1: "g", 2: "r", 3: "i"} for det in data["detections"]: band = fid_map.get(det["fid"], "?") print(f"{det['mjd']:.4f} {band} {det['magpsf']:.3f} ± {det['sigmapsf']:.3f} mag")
Download an LSST FITS stamp
from astropy.io import fits from io import BytesIO import requests resp = requests.get("https://api-lsst.alerce.online/stamps_api/stamp", params={ "oid": "2457648302395060224", "measurement_id": "7891234567890123456", "stamp_type": "cutoutDifference", "survey_id": "lsst", "file_format": "fits", }) hdul = fits.open(BytesIO(resp.content)) hdul.info()
